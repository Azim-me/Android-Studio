Textview & Edittext

To get text from Edittext and show Textview use this code

Note: Befor doing this make Edittext and Textview in XML file

First make a Textview and Edittext on XML

then make a variable declaration:

        
private TextView textview;
private EditText edittext;

Then find viewby id:

        
textview = findViewById(R.id.textView2);
edittext = findViewById(R.id.editTextText);

Get text from Edittext and show Textview:

        
String str = edittext.getText().toString();
textview.setText("Hello, " + str + "!");

Multiscreen & Intent

Intent: A mechanism in Android that enables communication between different components of an application or between different applications, facilitating actions and data transfer.

Types of Intents:

Explicit: Directly specify where you want to go within your app

Implicit: Let the system decide where to go based on the action you want to perform.
